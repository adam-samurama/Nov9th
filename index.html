User Characteristic Accessible design tip
1. Unable to see 1. Code all images with ALT text
2. Unable to perceive colors 2. Avoid using color alone to convey information
3. Unable to use mouse 3. Be sure all website features can be accessed using keyboard
4. Unable to hear 4. Add captions to multimedia
5. Prone to having seizures 5. Avoid content that flashes
6. Easily distractible 6. Keep the design simple
